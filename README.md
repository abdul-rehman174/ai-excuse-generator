# Goli Karao Pro

Describe a situation (type or speak it), slide the believability and drama dials, and get smart, tailored excuses — in your own language. Built as a single static HTML file; the AI runs on your own free Groq key.

## Features
- Type or speak your situation (voice input)
- Believability and drama sliders (1-10)
- Replies in the same language and style you write in
- Each excuse comes with a confidence rating and a delivery tip
- Installable PWA, works offline (the UI), no backend

## Run it
Open `index.html`, tap the gear, paste a free Groq API key from console.groq.com, and go. The key is stored only in your browser.
